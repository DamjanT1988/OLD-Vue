Om ramverket

Bootstrap är den mest populära CSS-ramverket med majoritet av marknadsandelar. Det är en gratis open-source front-end-utvecklingsramverk vilket är designad för responsiv utveckling genom mobile-first. Bootstrap ger bibliotek av syntax för olika ramverkdesign. Den innefatter bibliotek för HTMK, CSS och JS.
