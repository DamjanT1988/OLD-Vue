BOOTSTRAP

Om ramverket

Bootstrap är den mest populära CSS-ramverket med majoritet av marknadsandelar. Det är en gratis open-source front-end-utvecklingsramverk vilket är designad för responsiv utveckling genom mobile-first. Bootstrap ger bibliotek av syntax för olika ramverkdesign. Den innefatter bibliotek för HTMK, CSS och JS. Länk: https://getbootstrap.com/

Fördel & nackdel

Fördelar med Bootstrap är att det går snabbt att designa en HTML-sida genom en enkel CDN-länk. Vidare finns omfattande dokumentation med specifikationer som underlättar webbutveckling även med JS.

Nackdel är att det kan se liknande ut till andra webbplatser då det är så populärt.
