TAILWIND

Om ramverket

Tailwind är ett CSS-ramverk som är unikt somm enbart använder CSS som används i klasser i HTML-koden vilket minimerar CSS-kod som används. Likt andra ramverk underlättar det design och gör processen snabbare. Den använder JS för att läsa av HTML-filer för classer som används.
